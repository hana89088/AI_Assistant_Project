## 2024-06-23 - Serialize Broadcast Messages Once
**Learning:** In `src/python/main.py`, the `broadcast` method calls `json.dumps(message)` for every single client in the `self.clients` list inside the list comprehension for `asyncio.gather`. This means if there are 100 clients connected, it will serialize the exact same Python dictionary to JSON 100 times, performing redundant O(N) work instead of O(1).
**Action:** Extract the JSON serialization outside the list comprehension. Compute `json_message = json.dumps(message)` exactly once, and then send that pre-serialized string to all clients. This follows the codebase health standard memory directive: "extract JSON serialization out of broadcast loops to compute it exactly once to avoid redundant O(N) serialization overhead."
